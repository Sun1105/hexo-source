测试框架学习路线
│
├── 1. JUnit 5 基础
│     ├── 核心注解
│     │     ├── @Test
│     │     ├── @BeforeEach / @AfterEach
│     │     └── @BeforeAll / @AfterAll
│     ├── 断言方法
│     │     ├── assertEquals
│     │     ├── assertNotNull / assertNull
│     │     └── assertThrows
│     ├── 特性
│     │     ├── 参数化测试 (@ParameterizedTest)
│     │     └── 条件测试 (@EnabledIf / @DisabledIf)
│
├── 2. Mockito Mock 技术
│     ├── 基础
│     │     ├── @Mock / @InjectMocks
│     │     ├── when(), doReturn(), doThrow()
│     │     └── verify()
│     ├── 高级
│     │     ├── 参数匹配器 (any(), eq())
│     │     └── mockConstruction()
│     └── 与 Spring 集成
│           ├── @MockBean
│           └── @SpyBean
│
├── 3. Spring TestContext Framework
│     ├── 核心注解
│     │     ├── @SpringJUnitConfig
│     │     ├── @ContextConfiguration
│     │     ├── @DirtiesContext
│     │     └── @Sql
│     ├── 功能
│     │     ├── 上下文管理 (缓存)
│     │     ├── 依赖注入 (@Autowired)
│     │     └── 事务管理 (@Transactional)
│
├── 4. Spring Boot Test
│     ├── 核心注解
│     │     ├── @SpringBootTest
│     │     ├── @WebMvcTest
│     │     └── @DataJpaTest
│     ├── 测试工具
│     │     ├── OutputCaptureExtension
│     │     └── TestRestTemplate
│
├── 5. 高级技巧与最佳实践
│     ├── 上下文隔离 (@DirtiesContext)
│     ├── 测试性能优化 (上下文缓存)
│     ├── CI/CD 集成
│     └── Mock 与 Spring Boot 深度结合
│
└── 6. 自动化与覆盖率
      ├── 测试覆盖率工具 (JaCoCo, SonarQube)
      ├── 自动化测试 (Jenkins, GitHub Actions)
      └── 报告与分析 (HTML 报告)
