<span class="annotation">@RestController</span>
<span class="annotation">@RequiredArgsConstructor</span>
<span class="annotation">@Slf4j</span>
<span class="keyword">public class</span> <span class="function">TestController</span> <span class="keyword">extends</span> <span class="function">BaseController</span> {

    <span class="comment">/** APIのサービスクラス */</span>
    <span class="keyword">private final</span> TestService testService;

    <span class="comment">/**
     * APIのメインメソッド
     *
     * @param request  リクエスト情報
     * @param response レスポンス情報
     */</span>
    <span class="annotation">@PostMapping</span>(<span class="string">"/TEST0001"</span>)
    <span class="keyword">public void</span> <span class="function">execute</span>(HttpServletRequest request, HttpServletResponse response) {
        log.info(<span class="string">"TestController#execute() Start"</span>);

        <span class="comment">// リクエスト処理 → サービス呼び出し → レスポンス処理</span>
        responseProcess(
            response,
            testService.execute(requestProcess(request))
        );

        log.info(<span class="string">"TestController#execute() End"</span>);
    }
}</code></pre>

    <div class="section-title">2. 機能説明 (Functional Description)</div>
    <table>
        <tr>
            <th>物理名 / ID</th>
            <td>TestController / <span class="tag">TEST0001</span></td>
        </tr>
        <tr>
            <th>論理名</th>
            <td>テストデータ処理API</td>
        </tr>
        <tr>
            <th>処理概要</th>
            <td>クライアントからのリクエストを受け取り、業務ロジック（Service層）を呼び出した後、結果をレスポンスとして返却する。</td>
        </tr>
    </table>

    <div class="section-title">3. ロジック詳細 (Logic Details)</div>
    <div class="process-list">
        <strong>■ 実行フロー (Execution Flow)</strong>
        
        <ol>
            <li><strong>前処理ログ:</strong> 処理開始時にクラス名とメソッド名をログ出力する。</li>
            <li><strong>リクエスト解析 (requestProcess):</strong> 
                <ul>
                    <li><code>BaseController</code>から継承した共通メソッドを呼び出す。</li>
                    <li>HTTPリクエストをビジネスロジックで利用可能な形式に変換する。</li>
                </ul>
            </li>
            <li><strong>業務サービス実行 (testService.execute):</strong>
                <ul>
                    <li>インジェクトされた <code>TestService</code> を呼び出し、メインの業務処理を行う。</li>
                </ul>
            </li>
            <li><strong>レスポンス処理 (responseProcess):</strong>
                <ul>
                    <li>Service層の実行結果をHTTPレスポンスに書き込む。</li>
                </ul>
            </li>
            <li><strong>後処理ログ:</strong> 正常終了を確認するためのエンドログを出力する。</li>
        </ol>
    </div>

    <div class="section-title">4. 備考 (Remarks)</div>
    <ul>
        <li>本クラスは <code>@RequiredArgsConstructor</code> により、<code>TestService</code> のコンストラクタ注入を自動化している。</li>
        <li>例外処理は <code>BaseController</code> または <code>GlobalExceptionHandler</code> で一括管理することを前提とする。</li>
    </ul>
