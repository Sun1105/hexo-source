  <h1>.HTA 的简单明了介绍</h1>

  <hr />

  <h2>什么是 .HTA（HTML Application）</h2>
  <p>
    <strong>.HTA（HTML Application）是一种用 HTML + CSS + JavaScript 编写的 Windows 桌面应用程序格式。</strong>
    它由 Windows 的 <strong>mshta.exe</strong> 运行，因此可以像本地程序一样执行，而不是像普通网页那样受浏览器限制。
  </p>

  <hr />

  <h2>.HTA 的核心特点</h2>

  <h3>1. 用网页技术写桌面应用</h3>
  <ul>
    <li><strong>HTML：</strong>做界面</li>
    <li><strong>CSS：</strong>做样式</li>
    <li><strong>JavaScript：</strong>做逻辑</li>
    <li><strong>无需浏览器：</strong>直接由 <code>mshta.exe</code> 运行</li>
  </ul>

  <h3>2. 权限比普通网页高</h3>
  <p>HTA 可以：</p>
  <ul>
    <li><strong>读写本地文件</strong></li>
    <li><strong>调用 ActiveX</strong></li>
    <li><strong>操作注册表</strong></li>
    <li><strong>执行系统命令</strong></li>
  </ul>
  <p>这也是它常被用于内部工具或自动化脚本的原因。</p>

  <h3>3. 运行方式简单</h3>
  <p>双击 <code>.hta</code> 文件即可运行，就像 <code>.exe</code> 一样。</p>

  <hr />

  <h2>一个最简单的 HTA 示例</h2>
  <pre><code>&lt;html&gt;
&lt;head&gt;
  &lt;title&gt;Hello HTA&lt;/title&gt;
  &lt;HTA:APPLICATION
    ID="myApp"
    APPLICATIONNAME="HelloHTA"
    BORDER="thin"
    SCROLL="no"
  /&gt;
&lt;/head&gt;
&lt;body&gt;
  &lt;h1&gt;Hello, HTA!&lt;/h1&gt;
  &lt;button onclick="alert('按钮被点击')"&gt;点我&lt;/button&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre>
  <p>保存为 <code>hello.hta</code>，双击即可运行。</p>

  <hr />

  <h2>HTA 的常见用途</h2>
  <ul>
    <li><strong>内部工具：</strong>如批量处理、自动化脚本界面</li>
    <li><strong>简单桌面 GUI 程序</strong></li>
    <li><strong>系统管理工具</strong></li>
    <li><strong>旧系统的配置界面</strong></li>
  </ul>

  <hr />

  <h2>HTA 的缺点</h2>
  <ul>
    <li><strong>只适用于 Windows</strong></li>
    <li><strong>安全性较低：</strong>高权限 + ActiveX</li>
    <li><strong>技术较老：</strong>微软已不推荐使用</li>
  </ul>
