# 💻 电脑小技巧：如何将脚本固定到任务栏？

> **背景：** Windows 任务栏默认只允许固定 `.exe` 程序。如果你想把常用的 `.bat` 批处理或 `.hta` 应用固定到任务栏，直接拖拽是不行的。
> 
> **原理：** 通过修改快捷方式，让系统认为你在运行一个 `.exe`（如 cmd 或 mshta），从而实现“伪装固定”。

---

## ✅ 核心方案：用快捷方式“伪装”程序（推荐）

### 📌 适用类型
* **`.bat`** (批处理文件)
* **`.hta`** (HTML 应用程序)

---

## 🧭 操作步骤详解

### 第一步：创建初始快捷方式
1. 找到你的原始 `.bat` 或 `.hta` 文件。
2. **右键**点击该文件 -> 选择 **发送到** -> **桌面快捷方式**。

---

### 第二步：修改目标路径（关键）
右键点击桌面上刚生成的快捷方式，选择 **属性**，在“目标”栏中根据文件类型进行修改：

#### ▶ 情况 A：针对 `.bat` 文件
在原路径前添加 `cmd.exe /c`：
* **修改模板：** `C:\Windows\System32\cmd.exe /c "你的文件完整路径"`
* **示例：**
    ```bash
    C:\Windows\System32\cmd.exe /c "D:\Tools\backup_script.bat"
    ```

#### ▶ 情况 B：针对 `.hta` 文件
在原路径前添加 `mshta.exe`：
* **修改模板：** `C:\Windows\System32\mshta.exe "你的文件完整路径"`
* **示例：**
    ```bash
    C:\Windows\System32\mshta.exe "D:\MyApps\dashboard.hta"
    ```

---

### 第三步：美化图标（可选但推荐）
由于修改后图标会变成 CMD 的黑框，建议：
1. 在“属性”窗口点击 **更改图标**。
2. 选择一个你喜欢的图标（通常在 `shell32.dll` 或 `imageres.dll` 中有很多选择）。

---

### 第四步：固定到任务栏
* 最后，直接将这个修改好的桌面快捷方式 **拖拽到任务栏**，或者 **右键 -> 固定到任务栏** 即可成功！

---

## 🧩 技术小贴士 (Tips)

| 文件类型 | 执行核心 | 伪装逻辑 |
| :--- | :--- | :--- |
| **.bat** | `cmd.exe` | 显式调用命令提示符来运行脚本 |
| **.hta** | `mshta.exe` | 显式调用微软 HTML 宿主程序运行 |

> [!IMPORTANT]
> **为什么要这么做？**
> 因为 Windows 任务栏只识别 `.exe` 可执行文件。通过上述方法，我们本质上是在运行系统自带的 `.exe` 程序，只是顺便让它带上我们的脚本路径作为参数。
