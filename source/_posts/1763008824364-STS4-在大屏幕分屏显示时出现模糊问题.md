🧩 问题描述

在使用 Spring Tool Suite 4 (STS4) 时，
如果在 高分辨率大屏幕上进行分屏显示，界面可能会出现 字体模糊 或 界面不清晰 的情况。

✅ 解决办法：使用 Windows 的「系统」模式

右键点击 STS4 快捷方式 → 选择 属性

切换到 兼容性 选项卡

点击 更改高 DPI 设置

在弹出的窗口中，勾选：

✔️ 替代高 DPI 缩放行为
执行缩放的应用程序：系统（增强）

🪄 效果

此设置会让 Windows 负责缩放渲染，
通常可以获得更清晰的显示效果，
同时 不会让界面过大或比例异常。
